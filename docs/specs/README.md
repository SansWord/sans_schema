# Historical tier — specs

Per-milestone specs and brainstorms. **Allowed to go stale.** Each file is a
snapshot of the thinking at the time — *how we got here* — and is **not** the
source of truth.

- Don't cite these where they conflict with the code, [`../architecture.md`](../architecture.md),
  or the root `CLAUDE.md`.
- **Migrate before you retire:** before treating a milestone as done, move any
  still-current decision that lives *only* here into `../architecture.md` first.
- **Kept, never deleted** — the content may drift from the code, but the files
  stay forever as the record of *why* decisions were made.

Add new milestone specs alongside this README (e.g. `2026-08-first-gateway.md`).
